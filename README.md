# CalmMe – MCP Agent for Verisense

CalmMe is a CBT-style micro-intervention agent that interprets real-time heart rate (HR) and provides grounding, stabilisation, and emotional first-aid.

This project is **fully MCP-compliant** and can be registered directly in the Verisense Registry.

---

## Features
✔ MCP server (FastAPI)  
✔ Uses Ambient or Gemini LLMs  
✔ Real-time HR → Stress classification  
✔ CBT-based supportive micro-interventions  
✔ Fully compliant `.well-known/agent.json`
